# Realtime Chat App using Express.js and MySQL

![demo]

Ini adalah aplikasi real-time chat sederhana yang dibangun menggunakan Express.js untuk server dan MySQL sebagai database penyimpanan pesan.

## Fitur

- Pengguna dapat membuat atau bergabung ke dalam ruang obrolan (chat room).
- Pengguna dapat mengirim pesan ke ruang obrolan yang mereka ikuti.
- Pesan yang dikirim akan muncul secara real-time di layar pengguna lain yang tergabung dalam ruang obrolan yang sama.

## Instalasi

1. Pastikan Anda memiliki Node.js dan MySQL sudah terinstal di sistem Anda.
2. Clone repository ini ke dalam direktori lokal Anda.
3. Buka terminal atau command prompt, arahkan ke direktori proyek, dan jalankan perintah berikut untuk menginstal dependensi:

- tutorial clone app
    ```bash
    git clone https://github.com/syahrulangwar/realtimeChat-express-Js.git
      
- installasi untuk menjalankan aplikasi
   ```bash
   npm install express soket.io mysql 
   npm start
- pastikan untuk mengubah username dan password serta nama database sesuai dengan setup masing-masing
